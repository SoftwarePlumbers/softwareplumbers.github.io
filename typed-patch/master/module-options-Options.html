Class: Options

options~ Options

Holds options which change the way compare and patch operations are run.

An options object can be passed in to a compare or patch opertion in order to control how the operation is performed. However, it is more common to simply pass in a few paramters as a simple object which will then be automatically merged with the default options to create a new Options object.
