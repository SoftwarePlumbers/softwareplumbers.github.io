Class: ImmutableMap

ImmutableMap

Immutable Map extends AbstractMap to wrap an ES6 mutable map

new ImmutableMap(data)

Constructor

If a map is passed in, simply wraps that map (so the resulting immutable map is basically just a view on the parameter map). Otherwise, parameter is assumed to be an iterable over key/value pairs (i.e. [[k,v]...])

Parameters:
Name Type Description
data

Iterable over key/value pairs, Map, nothing (defauts to empty map)

Methods

builder()

Builder creates a mutable map as a copy of this map.

Builder is invoked by the set and delete operations of AbstractMap to create a mutable map for update.
